# 表情包萌和淘汰萌活动（618和1111）

## 活动开始前
1. app/Console/Commands/EmojiContestReset.php 清空旧数据(执行EmojiContest:reset)
2. Console/Commands/emojiContestData.json 准备好的表情包数据（后端用）
   resources/data/emojiContestData.ts  同样的数据复制一份（前端用）
3. app/Console/Commands/EmojiContestInject.php 注入新的表情包数据（EmojiContest:inject）
4. resources/vue/EmojiContest/EmojiContest.vue 这几个内容要改：
   1. 日期
   2. 调整要参赛的表情包
   3. 名称和代表图
   4. banner
   5. 投稿活动的链接
5. app/Http/Controllers/API/EmojiConstestController.php 要改：
   1. 日期
   2. 名称
6. 成就相关
   1. 准备好成就数据，前端medalData.ts，后端Medals.php
     - 表情包萌4个徽章：全员100票、全员1000票、冠军、淘汰、去年1111出道的角色
     - 出道萌1个徽章：全员100票、全员1000票、冠军
   2. app/Models/UserMedalRecord.php 更新成就的数据check_emoji_contest_total和check_emoji_contest_group
   3. app/Http/Controllers/API/EmojiConstestController.php 有一个角色成就的函数要开关

## 活动结束后
1. app/Console/Commands/EmojiContestMedal.php 要输入淘汰+冠军成就的组别id和徽章id，之后运行发放徽章(EmojiContest:medal)
